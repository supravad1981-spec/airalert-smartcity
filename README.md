## 📖 Overview

AirAlert SmartCity is a modern web application that demonstrates how Artificial Intelligence, IoT sensors, satellite imagery, and citizen reports can work together to monitor urban air pollution.

The project is designed as a responsive smart-city dashboard and can be hosted directly using **GitHub Pages**.

Clone the repository


Open the project folder

```bash

```

Run

Simply open

```
index.html
```

or deploy using **GitHub Pages**.
